# 49 Days — Navigation Notes

## 播放模式

影片共 **60 秒，分為 6 章**。可以像影片一樣從頭播到底（預設），也可以像簡報一樣跳章節。

| 章節 | 時間 | 標題 |
|---|---|---|
| 01 | 0–8s   | Origin · 起點 |
| 02 | 8–18s  | Opening · 首輪打擊 |
| 03 | 18–28s | Retaliation · 反擊 |
| 04 | 28–40s | Hormuz · 荷姆茲封鎖 |
| 05 | 40–50s | Toll · 代價 |
| 06 | 50–60s | Day 49 · 現況 |

## 快捷鍵

| 按鍵 | 行為 |
|---|---|
| **Space** | 播放 / 暫停 |
| **→** | 智慧下一步（見下方說明） |
| **←** | 智慧上一步 |
| **0 / Home** | 回到最開頭 |
| **點時間軸** | 跳到任意時間 |
| **點章節標籤** | 跳到該章節開頭 |

## 智慧 → / ← 邏輯

這不是單純的「前/後 1 秒」，而是簡報式的章節跳轉。

**按 →**：
- 若當前在**本章節的前 80%**（動畫還沒播完）→ 快速跳到本章節**結尾狀態**（所有元素都顯示完成那一刻）
- 若當前已接近**本章節結尾**（最後 20% 或更後）→ 跳到**下一章節開頭**（觸發入場動畫）

**按 ←**：對稱邏輯
- 前 20% → 跳到上一章節開頭
- 其他 → 跳到本章節開頭

按 → / ← 都會自動**暫停播放**，方便停下來看。再按 Space 繼續自動播放。

## 設計理念

- **Space = 影片模式**：從頭看到底
- **→ ← = 簡報模式**：一章一章推進，看完入場動畫不用等
- **時間軸拖曳 = 自由探索**

三種操作互不打架。

## 章節標籤列

時間軸上方有一條橫條，顯示 6 個章節名稱，當前章節會高亮並顯示本章節進度。點任一標籤可以直接跳到該章節開頭。
